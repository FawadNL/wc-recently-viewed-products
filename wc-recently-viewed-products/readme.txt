Show recently viewed products by user on your Woocommerce store.

== Description ==

Show recently viewed products by user on your Woocommerce store on product page, cart page, shop page based on setting and shortcode.

Use [rvps column="3" products="6"] 
 shortcode to show recently viewed products in pages, posts.

== Installation ==

Go to Plugins > Add new then upload and activate.

== Frequently Asked Questions ==

= How to install this plugin ? =

Go to Plugins > Add new then upload and activate.

= Where do it find plugin settings page? =

Settings page inde WooCommerce > WC Recently Viewed Products
